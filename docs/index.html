Computer Graphics Project Documentation

Introduction

This document presents the project for the Computer Graphics course at the University of Bologna for the academic year 2023/24. The project aims to create an interactive 3D application composed of at least one main polygonal object (using the Wavefront OBJ format). Additionally, appropriate scenography is required by lighting and texturing the scene objects.

Project Description

The main idea of the project is to create a small interactive parkour video game, where the main object is a ball that, through user commands using the keyboard, must collect the 3 coins scattered throughout the scenography. Consequently, the work was divided into two main parts: the creation or modification and subsequent rendering of the 3D objects using Blender and WebGL, and the development of the interaction logic with the user and objects (movements, ball rotation, jumping, collision with objects, etc...).

  • objs: folder containing all the polygonal objects, their textures, and the sounds used in the project;
  • src: the source code of the project written in Javascript.

3D Objects

The project renders 5 main polygonal objects, some of which have variations. All obj and mtl files of the objects are present in the objs folder.
Floor

The files for the floor are located in the floor folder. The object is a simple plane to which a grass texture has been applied using the grass.jpg file.

Tree

The tree object was downloaded from the website free3d.com, but the Decimate modifier was applied to reduce the complexity of the object (necessary to lower the file size). Additionally, the color of the applied material was modified to make it more visible in the scenography.

Cube
The cube object has 3 variants:
  • propic cube: a texture with my face was applied
  • glass cube: a transparent material was added
  • red cube: downloaded from free3d.com
Coin

The coin object was entirely created from scratch for the project. The uv-mapping functionality was used to generate the texture image, which was then colored as desired.

Ball

The ball object is a simple 3D sphere to which a texture is applied.

User Interface

The user interface consists of the canvas, the "Commands" section, and the settings section. The commands section not only displays the keys to move the ball but also makes them interactive on mobile devices through the use of touch events. Users can interact with the mouse to move the camera view or use the scroll wheel to adjust the FOV. In the settings section, users can modify the FOV, adjust the light positions, and enable or disable transparency and sounds. There is also a button to view the canvas in fullscreen mode. Additionally, it displays the number of collected coins and the time taken to collect all the coins. The application will show a modal windown for the loading fase and after the completion of the game.

Source code

Below are described the functionalities of the JavaScript modules and the main implemented features.
  • Modules
  • main.js
    Main module of the project. Download the models of the objects, initialize the objects and their positions. Initialize the scene, passing the position of all the cubes and coins present. Start the rendering process. 
  • scene.js
    Create the objects to handle the camera, lights, and the ball. Render all the objects in the scene. Create and render the Skybox. 
  • objLoader.js
    Parse the downloaded OBJ and MTL files.
     
  • objClass.js
    Manage the loading of the mesh and its rendering.
     
  • light.js
    Manage the position and settings of the lights.
     
  • const.js
    Contains default values for some settings and objects. Also includes the fragment shader and vertex shader used for the main scene. 
  • camera.js
    Manage the position and movements of the camera. Create the Shared Uniforms useful for rendering. 
  • ball.js
    Manage the position and physics of the ball's movements (velocity, acceleration, rotation, friction, jumping, gravity). Control and handle collisions with edges, cubes, and coins.
